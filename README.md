# Money Manager PWA (家計簿アプリ)

シンプルで高機能なオフライン対応の家計簿アプリです。
Progressive Web App (PWA) として設計されており、iPhoneやAndroidのホーム画面に追加してネイティブアプリのように使えます。

## 特徴
- **オフライン完全対応**: ネットワークがない場所でもサクサク動作します。
- **データは端末内に保存**: サーバーにデータを送信しないため、プライバシーも安心です。
- **iOS最適化**: iPhoneでの操作性を考慮したデザインとレイアウト。
- **バックアップ機能**: データをJSONファイルとして書き出し、機種変更時などに復元できます。

## 機能一覧

### 1. ダッシュボード (ホーム)
アプリを開くと最初に表示される画面です。
- **総資産**: 全ての口座の合計残高を一目で確認できます。
- **収支グラフ**: 今月の「収入」と「支出」のバランスを円グラフで可視化。
- **最近の取引**: 直近の入出金履歴リスト。
- **口座フィルタ**: 特定の口座（財布、銀行など）に絞り込んで表示できます。

### 2. 取引の追加
日々の収支を簡単に記録できます。
- **収入/支出** 切り替え
- **日付**: 過去の日付も選択可能。
- **金額**: 電卓不要でサッと入力。
- **カテゴリ**: 食費、交通費、給与などを選択。
- **口座**: どの口座からの出費かを選択。

### 3. 分析 (Analytics)
過去のデータを多角的に振り返ることができます。
- **月別レポート**: 左右の矢印で月を切り替え、過去の収支を確認。
- **カテゴリ別内訳**: 何にお金を使ったかがわかる円グラフ。
- **資産推移グラフ**: 過去6ヶ月の「総資産」の増減を折れ線グラフで表示。貯金のモチベーション向上に。
- **カレンダービュー**:
  - 日毎の「収入」「支出」をカレンダー形式で表示。
  - **収入**: 緑色、**支出**: 赤色で表示され、一目で収支がわかります。
  - 取引のない日は何も表示されず、スッキリと確認できます。
  - **詳細表示**: 日付をクリックすると、その日の取引詳細リストが表示されます。

### 4. 口座管理
複数の資産を一元管理できます。
- **口座一覧**: 各口座の現在の残高を確認。
- **新規追加**: 「財布」「銀行A」「貯金箱」など、自由に口座を追加可能。

### 5. 設定
アプリを自分好みにカスタマイズできます。
- **カテゴリ管理**: 収入・支出それぞれのカテゴリを自由に追加・削除。
- **テーマ切り替え**: 目に優しい「ダークモード」と、明るい「ライトモード」を切り替え可能。
- **データ管理**:
  - **JSONバックアップ**: 全データをJSON形式で保存・復元。
  - **CSVエクスポート**: ExcelやGoogle Sheetsで開ける形式（UTF-8 BOM付き）で全データを出力。確定申告や詳細分析に便利です。
  - **全データ削除**: アプリを初期状態にリセット。

## インストール方法 (iPhone/iPad)
1. Safariでこのページを開きます。
2. 画面下部の「共有」アイコン（四角から上矢印が出ているアイコン）をタップします。
3. メニューをスクロールして「ホーム画面に追加」を選択します。
4. 右上の「追加」をタップします。
5. ホーム画面にアイコンが追加され、アプリとして起動できます。
